# Date Range Picker

A robust, accessible, and timezone-aware Date Range Picker component.

## Features

- **Range Selection**: Intuitive start/end date picking.
- **Timezone Support**: DST-safe conversions.
- **Keyboard Navigation**: Full arrow-key support.
- **Accessibility**: ARIA grid semantics.
- **Constraints**: Min/max dates, blackout dates, and max range duration.

## Usage

Select a story from the sidebar to view different states:

- **Default**: Standard interactive picker.
- **KeyboardOnly**: Demonstration of focus management.
- **WithConstraints**: Validation logic (e.g. max 7 days).
- **TimezoneSwitch**: Testing DST transitions and zone changes.
